# NHL Tracker 6.9 visual-convergence audit

This release aligns the Home surface with the approved visual while retaining every existing destination and analytical capability.

## Changes

- Combined the desktop identity, season control, primary navigation, search and theme controls into one composed top bar.
- Reduced the Home opening to one clear editorial line: tonight first, then the whole season.
- Replaced the full slate duplication on Home with two featured, contextual matchup cards. The full authoritative slate remains in Tonight.
- Compressed the interactive Season File into a horizontal four-state dossier. Record, Form, Process and Path still change the displayed evidence and retain the rink/puck transition.
- Moved the visit briefing after the core analytical sequence so it no longer interrupts Tonight → Season → Analysis.
- Tightened the four-column Analysis Desk while preserving editing, ordering, hiding, saved players, pinned analytics and all deep links.
- Added file-edge tabs for Overview, Teams, Players, Metrics and Notes at large desktop sizes.
- Kept the existing responsive navigation and readable layouts for mobile and dark mode.

## Authoritative locations

- Home: concise orientation and selected signals.
- Tonight: complete current slate and game context.
- Schedule/Season: complete calendar and season navigation.
- Analysis destinations: full evidence, controls and saved exact views.
- Workspace: browser-local preferences, saved players, pins and remembered filters.

The design deliberately uses motion only for state change, hover feedback and navigation. Reduced-motion preferences remove those transitions.
